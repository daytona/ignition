Class Detail
Ignition.Model(methods)
Base Model class. This is used to create all Models for Ignition Objects. If you want, you can extend the base Model using jQuery.extend(). 
$i.m.MyModel = new Ignition.Model({
  load: function(callback) {
    // Load some content and peform the callback method
  },
  beforeSend: function() {
    // Overwriting the base Model's default beforeSend method
  }
});
// Calling:
$i.m.MyModel.load();
// Extend the base Model:
jQuery.extend(Ignition.Model.prototype, {
  myCustomMethod: function() {
    alert('This method will be available for all Models.');
  }
});
Parameters:
{Object} methods Optional
A collection of methods to extend the base Model with

json(options)
Get data from the specified URL as JSON data. This is a wrapper for jQuery.ajax(), with a pre-set dataType of 'json'. This method will use the callbacks you specify in the options, if any. If you haven't specified any such callbacks for this method, the default callbacks for the current Model object will be used, if available. Any additional options will be passed to the jQuery.ajax() function. $i.m.MyModel('/items.json', { success: function() { // Perform some actions when the XHR call is successful } });
